1️⃣ What is the difference between var, let, and const?



ans: var is the old way. It can cause problems. By var variable name can be duplicate

let is the new way to make a variable that you want to change later. (Example: let score = 10; then later score = 20;)
const is for things that should NOT change. (Example: const myName = "John"; You cannot change myName later.)
